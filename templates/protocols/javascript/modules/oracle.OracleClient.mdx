# Class: OracleClient

[oracle](/templates/protocols/javascript/modules/oracle).OracleClient

Client is a client for Oracle database.
Internally client uses go-ora driver.

**`Example`**

```javascript
const oracle = require('nuclei/oracle');
const client = new oracle.OracleClient();
```

### ExecuteQuery

▸ **ExecuteQuery**(`host`, `port`, `username`, `password`, `dbName`, `query`): [`SQLResult`](/templates/protocols/javascript/modules/oracle.SQLResult)

ExecuteQuery connects to Oracle database using given credentials and executes a query.
It returns the results of the query or an error if something goes wrong.

#### Parameters

| Name | Type |
| :------ | :------ |
| `host` | `string` |
| `port` | `number` |
| `username` | `string` |
| `password` | `string` |
| `dbName` | `string` |
| `query` | `string` |

#### Returns

[`SQLResult`](/templates/protocols/javascript/modules/oracle.SQLResult)

**`Example`**

```javascript
const oracle = require('nuclei/oracle');
const client = new oracle.OracleClient();
const result = client.ExecuteQuery('acme.com', 1521, 'username', 'password', 'XE', 'SELECT * FROM dual');
log(to_json(result));
```
